Kruskal's Algorithm Demo (June 2020-Present)
- Web application built in C# which allows users to visualize kruskal's algorithm finding a minimum spanning tree.
- Features an interactable UI built in Unity allowing users to draw their own connected weighted graphs.
- Created using an object-oriented approach, enabling data stucture implementation such as heap, set, and union-find.
- Utilizes a minimum heap implemented as an array to determine which weighted edge should be analyzed next.
- Other features are still in progress, such as weight designation and visuals, and a step-by-step demonstration of kruskal's algorithm.
